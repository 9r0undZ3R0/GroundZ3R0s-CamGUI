# GroundZ3R0's CamGUI

## A Gui Camera Mod that adds camera positions Another Axiom didn't!

- Shift + Left Click: Move CamGUI
- Shift + Right Click: Move Spectator
- `: Toggle CamGUI
- Tab: Toggle Spectator
